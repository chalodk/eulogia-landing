# eulogia-landing
Landing page de Eulogia.cl

## Descripción

Landing page minimalista y moderna para Eulogia.cl con un diseño limpio y fresco.

## Características

- **Hero Section**: Sección principal con slogan, tagline y botón de contacto sobre imagen de banner
- **Features Section**: Sección de características con imagen vertical y 3 puntos destacados
- **Diseño Responsive**: Adaptable a móviles, tablets y escritorio
- **Estilo Minimalista**: Colores suaves, tipografía moderna y transiciones suaves

## Uso

Simplemente abre `index.html` en tu navegador o despliega los archivos en tu servidor web.

## Personalización

Para personalizar el contenido:

1. **Slogan y tagline**: Edita el texto en la sección `.hero-content` del archivo `index.html`
2. **Bullet points**: Modifica el contenido de las características en la sección `.features-list`
3. **Colores**: Ajusta los colores en `styles.css` modificando las variables de color
4. **Imágenes**: Las imágenes se cargan desde Supabase storage
